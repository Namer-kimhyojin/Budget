# 2026년도 예산 수립 가이드라인 및 기준

## 1. 주요 용어 정의

| 용어 | 주요 내용 | 비고 |
| :--- | :--- | :--- |
| **수입예산** | 매 회계연도 지출의 재원이 되는 모든 현금적 수입 | |
| **지출예산** | 매 회계연도에 법인의 목적을 수행하기 위한 일체의 지출 | |
| **본예산** | 매 회계연도마다 예산안을 편성하여, 이사회 심의·의결하여 성립된 당초의 예산 | |
| **추가경정예산** | 본예산 성립 후 변경 사유 발생 시 임시이사회 승인을 받은 예산 (정리추경: 최종 예산서) | |
| **추가경정 성립 전 사용승인** | 용도가 정해진 경비(국고보조금 등)로 추경 성립 전 원장 승인 후 사용 | |
| **명시이월** | 연도 내 지출 불가 예측 시 예산서 명시 및 사전 승인 후 다음연도 이월 | |
| **사고이월** | 지출원인행위 후 불가피한 사유로 연도 내 지출 못한 경비 (엄격 제한) | |
| **계속비 이월** | 수년도 사업 경비, 완성연도까지 이월 사용 | |

## 4. 계속사업 이월액 편성 원칙

*   **2025년 정리추경 예산서 상 "차년도 이월금"**
    *   → **2026년 본예산 "계속사업 이월금"**으로 편성 필요 (수입)
*   **2026년 예산서의 "차년도 이월금"**
    *   → **2027년으로 이월**하여 집행하는 예산
    *   ⚠️ **차년도 이월금으로 편성한 예산은 해당연도(2026년)에 집행불가**
*   **2026년 수입 = 계속사업 이월금 + 당해연도 예산 + 차년도 이월금**

## 5. 산출내역 작성 양식

시스템 입력 시 다음 형식을 준수:
*   **[구분] (재원) 산출내역**
    *   예시 1: `[계속사업 이월] (국비) (중소벤처기업부) ...`
    *   예시 2: `[당해연도] (시비) (포항시) ...`
    *   예시 3: `[차년도 이월] (시비) (포항시) ...`
*   **계산식:** `단가(원) × 수량(명/식) × 횟수(월/회) = 합계`
